== Installation ==

Installing this plugin as regular wordpress plugin.

After install, this plugin enable a custom post called <strong>U Hover Effects</strong>. Add category & add images using Add More Hover Item on that custom post. 
After adding items, first you need enter your category on shortcode generator. Use category title, not category id or slug. Make sure spelling is ok. It is case sensitive. You can customize the plugin with plugin settings under the post, after customizing click insert and save the post. copy shortcode from sidebar and you are ready to use this plugin in your post, page. 

Go to page > Add New. On the editor, just paste the shortcode.
